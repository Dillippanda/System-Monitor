# System Monitor

A terminal-based system monitoring tool built in C++ that displays real-time system information using ncurses.

## Features

- CPU usage monitoring
- Memory usage statistics
- Process information
- System information display

## Prerequisites

- C++ compiler with C++17 support
- ncurses library
- Make build system

## Building

To build the project:

```bash
make
```

To clean build files:

```bash
make clean
```

## Usage

Run the system monitor:

```bash
./sysmon
```

## Project Structure

- `src/` - Source files
- `include/` - Header files
- `Makefile` - Build configuration
